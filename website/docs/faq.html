Frequently Asked Questions

Common questions about BulkBlast functionality, safety, and fees.

What assets can I send with BulkBlast?

BulkBlast supports the native Solana token (SOL), all standard SPL tokens (like USDC, BONK, WIF), and Metaplex standard NFTs.

Is BulkBlast mobile-only?

Yes, BulkBlast is designed as a mobile-first application specifically for Android. We believe the future of crypto adoption lies in powerful mobile tools. An iOS version is planned for the future.

What networks are supported?

BulkBlast supports both Solana Mainnet-Beta (for real transactions) and Devnet (for testing). You can switch networks easily in the Settings tab.

Why do fees use SKR?

Using SEEKER (SKR) allows us to align with the Solana Mobile ecosystem and offer native discounts to mobile users. It simplifies fee processing and supports the community.

How does the Solana Mobile discount work?

If you are using a Solana Mobile device (Saga or Seeker), the app automatically detects this and applies a 50% discount to all BulkBlast service fees.

Is BulkBlast safe to use?

Yes. BulkBlast is a non-custodial tool, meaning we never have access to your private keys or funds. All transactions are simulated before signing to ensure safety. We recommend testing on Devnet first if you are unsure.

Are swaps done inside the app?

Yes, BulkBlast integrates Jupiter Aggregator directly, allowing you to swap assets without leaving the app. This is perfect for quickly acquiring SKR for fees.

Is documentation still evolving?

Yes, as BulkBlast adds new features (like improved NFT support, new routing partners, etc.), this documentation will be updated. Check back often for the latest guides.
