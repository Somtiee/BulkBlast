NFT Support

Bulk send digital collectibles, whitelists, and rewards with full NFT support on Solana.

Sending NFTs

BulkBlast isn't just for tokens. We provide a seamless workflow for distributing NFTs to multiple recipients.

Supported Assets:

  • Metaplex Standard NFTs: The most common NFT type on Solana.
  • Digital Collectibles: Game items, art, memberships.
  • Compressed NFTs (cNFTs): (Planned for future updates)

Whole Numbers Only

Unlike fungible tokens (like SOL or USDC), NFTs are non-fungible. You can only send whole numbers (e.g., 1 NFT, 5 NFTs). Decimals are not supported.

Grouped Collections

BulkBlast automatically groups your NFTs by collection, making it easy to:

  • Select multiple items from the same series.
  • Distribute rewards to specific holders.
  • manage large inventories efficiently.

Safety Checks

Sending NFTs requires extra care. BulkBlast includes specific safeguards:

  • Collection Verification: Ensures you are sending the correct assets.
  • Fail-Closed Logic: Unsupported or unrecognized token standards will prevent the transaction, protecting you from errors.
  • Recipient Validation: Checks that destination wallets can receive the asset type.

As Solana's NFT standards evolve (e.g., Token 2022, Core), BulkBlast will continue to update its support to include the latest innovations.
