Getting Started

Set up BulkBlast on Android and prepare for your first launch or bulk distribution flow.

Android Only

BulkBlast is currently available exclusively for Android. iOS support is planned for future roadmap phases.

Quick Start Checklist

  1. Install the App Download the APK from the official website or GitHub Releases.
  2. Set Up Wallet Create a new wallet or import an existing private key.
  3. Add Recipients Import a CSV or manually add wallet addresses.
  4. Select Asset Choose SOL, an SPL token, or NFT.
  5. Choose Fee Mode Pay service fees in SKR or SOL.
  6. Review & Send Run the safety simulation and confirm execution.

Optional after your first blast: Launch with BAGS

You can optionally use BulkBlast to launch a token via BAGS and then jump into distribution.

  1. Open Launch and configure your token.
  2. Launch via BAGS in-app.
  3. Tap BLAST NOW to hand off to Create Drop.
  4. This handoff path is eligible for a one-time free blast fee.

1. Installation

BulkBlast is distributed as an APK file for Android devices. To install:

  1. Download the latest bulkblast.apk from the website or GitHub.
  2. Open the file on your Android device.
  3. If prompted, allow installation from "Unknown Sources" (this is standard for direct APK downloads).
  4. Launch the app.

2. Wallet Setup

When you first launch BulkBlast, you'll need to connect a wallet. BulkBlast supports:

  • Create New Wallet: Generates a new non-custodial Solana wallet locally on your device.
  • Import Private Key: Import an existing wallet using its private key.

Security Note

Your private keys are encrypted and stored securely on your device's keystore. They are never transmitted to any external server.

3. Devnet vs Mainnet

You can switch between Solana networks in the Settings tab:

  • Mainnet-Beta: For real transactions with real assets.
  • Devnet: For testing with "fake" SOL. We highly recommend trying your first bulk send on Devnet to get comfortable with the workflow.
