How It Works

A step-by-step guide to executing bulk transactions, airdrops, and asset distributions with BulkBlast.

Step 1: Add Recipients

Start by selecting the wallet addresses you want to send assets to. BulkBlast provides several ways to import recipients:

CSV Import

Import a CSV file containing wallet addresses directly from your phone's storage.

Manual Entry

Type or paste individual wallet addresses one by one.

Scan QR Code

Use your phone's camera to scan a wallet address QR code.

Random Selection

For giveaways, randomly select winners from a list.

Step 2: Choose Asset

After adding recipients, select the asset you want to distribute. BulkBlast supports:

  • SOL: The native token of the Solana blockchain.
  • SPL Tokens: Any standard Solana token (USDC, BONK, WIF, custom tokens).
  • NFTs: Select one or more NFTs to send to individual recipients.

Token Selection

If you don't see your token, ensure you have a balance in your wallet. Custom token mint addresses can also be pasted directly.

Step 3: Set Amount

Define how much each recipient receives:

  • Equal Amount: Send the same amount (e.g., 1 SOL) to every wallet.
  • Specific Amounts: Specify different amounts for each recipient (supported via CSV import).
  • NFT Quantities: Must be whole numbers (1, 2, etc.).

Step 4: Review & Simulate

Before any transaction is signed, BulkBlast performs a safety check:

  • Simulation: Simulates the transaction on the blockchain to ensure it will succeed.
  • Spend Caps: Checks if the total amount exceeds your set limits.
  • Fee Calculation: Shows the network fee and any BulkBlast service fee upfront.
  • Balance Check: Verifies you have enough funds to cover the transfer + fees.

Step 5: Send

Confirm the transaction to start the "blast."

  • Batching: Transactions are sent in efficient batches to optimize speed and cost.
  • Progress UI: Watch the progress bar as transactions confirm on-chain.
  • Retry: If any transaction fails (e.g., due to network congestion), you can retry only the failed ones.

History & Receipts

Once completed, the transaction is saved to your History tab. You can view details, see the transaction signature, and export a receipt for your records.
