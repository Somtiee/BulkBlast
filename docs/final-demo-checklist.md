# Final Demo Flow Checklist

Use this script when recording a video demo or presenting the app to judges/reviewers.

## 1. Opening & Wallet
- [ ] Launch App -> See Splash Animation.
- [ ] Land on "Create Drop" (Home).
- [ ] Tap "Wallet" toggle to show portfolio view.
- [ ] Show balances loading (SOL, USDC, etc.).

## 2. Preparing a Drop
- [ ] Tap "Create Drop".
- [ ] **Select Asset**: Choose SOL (or a token).
- [ ] **Add Recipients**:
    - Tap "Manual" -> Enter a known address (e.g., your own second wallet).
    - OR Tap "CSV" -> Pick a sample `.csv` file.
- [ ] **Review**: Tap "Next". Show the "Safety Check" spinner running and passing.

## 3. Execution
- [ ] Tap "Execute Airdrop".
- [ ] Show the progress screen.
- [ ] Wait for "Success" checkmark.
- [ ] Tap "View Receipt" or "Done".

## 4. History & Management
- [ ] Go to "History" tab.
- [ ] Show the drop that just finished at the top.
- [ ] Tap it to see details.
- [ ] Tap "Export CSV" to show the share sheet.

## 5. Swap (The "Power User" Feature)
- [ ] Go back to Home.
- [ ] Tap the "Swap" icon (or "Swap" button in wallet).
- [ ] Enter `0.1 SOL`.
- [ ] See the quote load (Powered by Jupiter).
- [ ] (Optional) Execute the swap if you have funds, or just show the UI.

## 6. Settings & Polish
- [ ] Go to "Settings".
- [ ] Toggle "Dark Mode" on/off to show theme support.
- [ ] Scroll down to show "Network: Mainnet" and "Diagnostics" (showing API keys present).
