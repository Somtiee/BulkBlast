## Full Description

**BulkBlast** is the ultimate mobile-first tool for managing Solana airdrops and bulk transactions directly from your phone. Designed for community managers, NFT creators, and power users, BulkBlast makes sending assets to hundreds of recipients as easy as sending a text.

### 🚀 Key Features

*   **Bulk Sender**: Send SOL, SPL tokens, and NFTs to multiple addresses in a single transaction flow.
*   **CSV Import**: Import recipient lists from CSV files or paste them directly. No more manual entry errors.
*   **Safety First**: Automatic address validation and "Safety Check" simulations ensure you never burn assets on failed transactions.
*   **Built-in Swap**: Powered by **Jupiter**, swap assets instantly without leaving the app to prepare for your drop.
*   **Detailed History**: Track every batch transaction with exportable CSV receipts for your records.
*   **Secure**: Your keys never leave your device. Non-custodial and secure by design.

### 📱 Optimized for Solana Mobile
BulkBlast is built from the ground up for the Solana Mobile Stack.
*   **Saga Ready**: Perfectly scaled UI for the Saga device.
*   **Seeker Discount**: Exclusive fee discounts for Solana Mobile users.

*   Distributing DAO rewards.
*   Airdropping NFT whitelist tokens.
*   Paying contributors in bulk.
*   Sending payments to friends split across multiple transactions.
